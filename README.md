# Fundamentos y Técnicas de Seguridad para Aplicaciones

Repositorio para la clase «Fundamentos y Técnicas de Seguridad para
Aplicaciones» de la Maestría del [Posgrado en Ciencias e Ingeniería de la
Computación](https://pcic.unam.mx/) de la UNAM.

En este repositorio daremos seguimiento a las actividades del cursado de la
materia, particularmente:

- [**Material**](./material/README.md): Material de cursado: Los temas
  conforme se van presentando, las lecturas que vamos abordando, etc.
- [**Entregas**](./entregas/README.md): Las entregas relativas a tareas,
  ejercicios, o diversas consignas del cursado.
